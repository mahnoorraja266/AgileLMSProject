# Academic Portal LMS Dashboard

A beautiful, premium, and fully responsive student learning management system (LMS) dashboard built with Next.js, TailwindCSS, and Shadcn UI.

## Features

- **Personalized Dashboard Hub**: Profile overview, degree progress tracking, cumulative GPA statistics, achievements, and real-time academic announcements.
- **My Courses & Course Details**: Interactive courses overview with detail panel viewing syllabus, structure, and lessons.
- **Assignments Tracker**: Visual assignment status, due dates, submission portal, and grading status.
- **Grades Panel**: Detailed course grades, breakdown of assignments, quizzes, mid-terms, and cumulative scores.
- **Calendar Scheduling**: Integrated academic calendar showing events, class schedules, and deadlines.
- **Direct Messaging**: Chat platform for communicating with class instructors and peers.
- **Learning Resources**: Student library, research portal, utility links, and documents download.
- **Certifications & Achievements**: Dean's list, certified badges, course completion certificates.
- **Theme & Profile Settings**: Comprehensive settings panel for profile updates, notifications, and dark/light mode toggle.

## Tech Stack

- **Framework**: Next.js (React 19)
- **Styling**: Tailwind CSS
- **UI Components**: Shadcn UI & Radix Primitives
- **Icons**: Lucide React & Material Symbols
- **Visuals**: Recharts (for performance visualization), Embla Carousel

## Installation & Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build production bundle:
   ```bash
   npm run build
   ```

4. Start production server:
   ```bash
   npm start
   ```
